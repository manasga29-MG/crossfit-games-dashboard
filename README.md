# CrossFit Games 2019 Dashboard

## Project Overview

This project analyzes athlete performance in CrossFit Games 2019 using Power BI.

## Business Objectives

* Analyze athlete demographics
* Identify top performers
* Compare countries
* Evaluate workout difficulty

## Dashboard Pages

1. Overview
2. Athlete Performance
3. Demographics
4. Workout Analysis

## Tools Used

* Power BI
* Excel
* GitHub

## Key Insights

* Most athletes are between 25–34 years old.
* USA has highest participation.
* Top athletes show consistent performance.
